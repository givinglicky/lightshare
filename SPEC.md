## 1. 專案概述

### 1.1 願景

陽光種子希望園是一個溫暖、互助的社區平台。我們相信每一份小小的善意都像是一顆種子，只要有陽光的照耀與細心的呵護，終將在心田開出希望的花朵。

### 1.2 使命

- 連結鄰里情感，打造互助社區
- 提供即時的求助與支援管道
- 分享生活中的正能量故事

### 1.3 目標使用者

- 社區居民
- 想要尋求鄰里幫助的人
- 願意分享正向故事和提供支持的人

## 3. 技術架構

### 3.1 前端

- **框架**: React 19
- **語言**: TypeScript ~5.8.2
- **建構工具**: Vite 6.2.0
- **CSS**: Tailwind CSS 4.1.14
- **路由**: React Router DOM 7.13.1
- **動畫**: Motion 12.23.24

### 3.2 後端

- **框架**: Express 4.21.2 (已安裝，待整合)
- **資料庫**: SQLite (better-sqlite3 12.4.1)
- **認證**: JWT + bcryptjs

### 3.3 部署

- **前端**: Vercel / Zeabur
- **後端**: Zeabur (已配置)
