# 陽光種子希望園 (LightShare)

陽光種子希望園是一個溫暖、互助的社區平台。我們相信每一份小小的善意都像是一顆種子，只要有陽光的照耀與細心的呵護，終將在心田開出希望的花朵。

## 專案目標
- 連結鄰里情感，打造互助社區。
- 提供即時的求助與支援管道。
- 分享生活中的正能量故事。

## 技術棧
- React 19
- TypeScript
- Tailwind CSS
- Motion (動畫)
- Vite
