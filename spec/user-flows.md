# LightShare 使用者流程

## 1. 訪客流程

### 1.1 首次訪問

```
[訪問網站] → [首頁/About] → [瀏覽公開內容] → [點擊註冊/登入]
```

**步驟說明**:

1. 訪客訪問網站
2. 看到關於我們頁面或公開的社群動態
3. 點擊「註冊」或「登入」按鈕

### 1.2 註冊流程

```
[註冊頁面] → [填寫表單] → [驗證提交] → [註冊成功] → [自動登入] → [Feed]
```

**詳細步驟**:

1. 訪客進入註冊頁面 (`/register`)
2. 填寫:
   - 姓名
   - Email
   - 密碼
   - 確認密碼
3. 點擊「註冊」按鈕
4. 系統驗證資料 (格式、唯一性)
5. 註冊成功，自動登入
6. 跳轉至社群動態頁面 (`/feed`)

### 1.3 登入流程

```
[登入頁面] → [填寫帳號密碼] → [驗證] → [登入成功] → [Feed]
```

**詳細步驟**:

1. 使用者進入登入頁面 (`/login`)
2. 填寫:
   - Email
   - 密碼
3. 點擊「登入」按�道
4. 系統驗證帳號密碼
5. 登入成功，產生 JWT token
6. 跳轉至社群動態頁面 (`/feed`)

## 2. 已登入使用者流程

### 2.1 瀏覽動態 (首頁)

```
[Feed 頁面] → [滾動瀏覽貼文] → [點擊感興趣的貼文]
```

**詳細步驟**:

1. 使用者位於社群動態頁面 (`/feed`)
2. 滾動瀏覽貼文列表
3. 點擊有興趣的貼文
4. 進入貼文詳情頁面 (`/post/:id`)

### 2.2 發佈貼文

```
[點擊發佈按鈕] → [選擇類型] → [編輯內容] → [預覽] → [發佈] → [成功頁面]
```

**詳細步驟**:

1. 點擊浮動「發佈」按鈕
2. 進入發佈頁面 (`/create`)
3. **步驟 1**: 選擇類型
   - 求助分享
   - 故事分享
4. **步驟 2**: 編輯內容
   - 填寫標題
   - 填寫內容
   - 選擇分類標籤
5. **步驟 3**: 預覽確認
6. 點擊「發佈」按鈕
7. 進入成功頁面 (`/success`)
8. 可選擇返回 Feed 或查看自己的貼文

### 2.3 互動流程

#### 2.3.1 按讚

```
[在 Feed/詳情頁] → [點擊愛心圖標] → [愛心變色] → [數量+1]
```

**詳細步驟**:

1. 在貼文卡片或詳情頁
2. 點擊愛心圖標
3. 圖標變為紅色filled
4. 按讚數增加 1

#### 2.3.2 收藏

```
[在 Feed/詳情頁] → [點擊書籌圖標] → [書籌變色]
```

**詳細步驟**:

1. 在貼文卡片或詳情頁
2. 點擊書籌圖標
3. 圖標變為filled
4. 收藏成功

#### 2.3.3 留言

```
[在詳情頁] → [輸入留言] → [點擊發送] → [留言顯示]
```

**詳細步驟**:

1. 在貼文詳情頁
2. 滑至留言區塊
3. 輸入留言內容
4. 點擊「發送」按鈕
5. 留言立即顯示在列表中

### 2.4 個人檔案流程

#### 2.4.1 查看個人檔案

```
[點擊頭像/個人] → [個人檔案頁面] → [查看我的貼文/收藏]
```

**詳細步驟**:

1. 點擊導航欄頭像
2. 進入個人檔案頁面 (`/profile`)
3. 預設顯示「我的貼文」標籤
4. 可切換至「收藏」標籤查看收藏的貼文

#### 2.4.2 編輯個人資料

```
[個人檔案] → [點擊編輯] → [編輯資料頁面] → [修改資料] → [儲存]
```

**詳細步驟**:

1. 在個人檔案頁面
2. 點擊「編輯檔案」按鈕
3. 進入編輯頁面 (`/profile/edit`)
4. 修改名稱、bio 等資訊
5. 點擊「儲存」按鈕
6. 返回個人檔案頁面，顯示更新後的資料

### 2.5 通知流程

#### 2.5.1 查看通知

```
[點擊通知圖標] → [通知面板彈出] → [點擊通知項目]
```

**詳細步驟**:

1. 點擊導航欄通知圖標
2. 彈出通知面板
3. 顯示最近的通知列表
4. 點擊通知項目可跳轉至相關貼文

#### 2.5.2 通知設定

```
[個人檔案] → [通知設定] → [設定頁面] → [調整開關] → [儲存]
```

**詳細步驟**:

1. 在個人檔案頁面
2. 點擊「通知設定」
3. 進入通知設定頁面 (`/profile/notifications`)
4. 調整各類通知的開關狀態
5. 點擊「儲存」

### 2.6 搜尋流程

```
[點擊搜尋圖標] → [輸入關鍵字] → [顯示結果] → [點擊結果]
```

**詳細步驟**:

1. 點擊導航欄搜尋圖標
2. 進入搜尋頁面 (`/search`)
3. 輸入搜尋關鍵字
4. 顯示符合的貼文列表
5. 點擊有興趣的貼文

## 3. 登出流程

```
[點擊登出] → [確認登出] → [清除 Token] → [跳轉至登入頁]
```

## 4. 錯誤處理流程

### 4.1 登入失敗

```
[輸入錯誤帳號密碼] → [點擊登入] → [顯示錯誤訊息] → [停留在登入頁]
```

### 4.2 發佈失敗

```
[發佈貼文] → [伺服器錯誤] → [顯示錯誤訊息] → [保留輸入內容]
```

### 4.3 權限不足

```
[訪問需要權限的頁面] → [驗證失敗] → [跳轉至登入頁]
```
