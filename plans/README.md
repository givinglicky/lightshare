## 貢獻指南

在建立新計劃時，請遵循以下格式：

```
plans/
├── README.md
├── YYYY-MM-DD-plan-name.md
└── ...
```

計劃文件應包含：
- 計劃名稱與目標
- 具體任務清單
- 時間估計
- 依賴關係
